A minified and customized fork of Helix. The idea is to remove everything I personally don't use.
Feel free to use, but I advise you to go with the [official version](https://helix-editor.com/).

# Changes

Language support only for:

- bash
- css
- env
- html ([superhtml](https://github.com/kristoff-it/superhtml))
- java
- javascript
- jsdoc
- json
- jsonc
- markdown
- odin
- python
- rust
- scss
- sql
- typescript
- toml
- zig

Themes:

- ao
- base16
- default
- [everblush](https://everblush.github.io)

# Contributing

This is a customized and minified version of Helix, tailored to myself. For that reason I don't accept
PRs. Contributions should be made to the official Helix repository [here](https://github.com/helix-editor/helix).
